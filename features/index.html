Features

A modular theme for Kirby CMS

Options
Add or remove any widget: title, subtitle, author, date, subpages, etc.

Shortcodes
Customize text content with shortcodes for columns, buttons, separators, and custom text.

Layout
Multiple layouts. Layouts can be remixed to create new combinations.

Search
Search content by field or query with a search widget or query string.

Colors
Set custom colors for links, decorators, and primary color with color field.

Navbar
Navbar with custom links, sticky navbar, dropdown menus, custom labels.

External files
Add custom fonts, or link to CDN libraries via URL form panel.

Sidebar
Global sidebar with support for search, recent pages, related pages, tags.

Custom subpages
Load subpages from any page. For example: make Home load subpages from Blog.

Tags
Add tags to categorize content. Pages with same tags will be grouped as related pages.

Contact form
Add a working contact form to any page.

OpenGraph
Automatic preview generation for OpenGraph based on page content.

Custom fonts
Link to custom fonts via URL. Or upload your own fonts and link via CSS field.

Dark mode
Automatically change to dark mode when user has requested dark mode at system level.

Browser support
This theme uses css grid, flexbox, and custom properties, browser support extends accordingly.
